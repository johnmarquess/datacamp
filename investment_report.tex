\subsubsection{Investment Annual
Summary}\label{investment-annual-summary}}

The \texttt{investment\_annual\_summary} dataset provides a summary of
the dollars in millions provided to each region for each fiscal year,
from 2012 to 2018.

\hypertarget{investment-projects-in-bangladesh}{%
\subsubsection{Investment Projects in
Bangladesh}\label{investment-projects-in-bangladesh}}

The \texttt{investment\_services\_projects} dataset provides information
about each investment project from 2012 to 2018. Information listed
includes the project name, company name, sector, project status, and
investment amounts. Projects that do not have an associated investment
amount are excluded from the plot.

\begin{Shaded}
\begin{Highlighting}[]
\NormalTok{country\_investment\_projects }\OtherTok{\textless{}{-}}\NormalTok{ investment\_services\_projects }\SpecialCharTok{\%\textgreater{}\%}
  \FunctionTok{filter}\NormalTok{(country }\SpecialCharTok{==}\NormalTok{ params}\SpecialCharTok{$}\NormalTok{country) }

\FunctionTok{ggplot}\NormalTok{(country\_investment\_projects, }\FunctionTok{aes}\NormalTok{(}\AttributeTok{x =}\NormalTok{ date\_disclosed, }\AttributeTok{y =}\NormalTok{ total\_investment, }\AttributeTok{color =}\NormalTok{ status)) }\SpecialCharTok{+}
  \FunctionTok{geom\_point}\NormalTok{() }\SpecialCharTok{+}
  \FunctionTok{labs}\NormalTok{(}
    \AttributeTok{title =} \StringTok{"Investment Services Projects"}\NormalTok{,}
    \AttributeTok{x =} \StringTok{"Date Disclosed"}\NormalTok{,}
    \AttributeTok{y =} \StringTok{"Total IFC Investment in Dollars in Millions"}
\NormalTok{  )}
\end{Highlighting}
\end{Shaded}
